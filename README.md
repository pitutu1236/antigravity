# Antigravity - Trailers de las 100 Mejores Películas del Siglo XXI

Web estática para mostrar y reproducir trailers de 100 películas conforme a la lista de The New York Times.

## Cómo publicar en GitHub Pages

1. Crea un repositorio público en GitHub.
2. Sube estos archivos al repositorio.
3. En GitHub, ve a `Settings` > `Pages`.
4. Selecciona la rama `main` y la carpeta `/ (root)`.
5. Guarda los cambios.
6. Accede a la URL provista por GitHub Pages.

## Archivos principales

- `index.html`
- `style.css`
- `app.js`
- `movies_data.json`

## Notas

- La app carga `movies_data.json` mediante `fetch`, por lo que debe estar publicada en un servidor HTTP.
- Las rutas son relativas y son compatibles con GitHub Pages.
